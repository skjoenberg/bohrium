Developer Guide
---------------

Bohrium is hosted and made publicly available via a `git-repository <https://github.com/bh107/bohrium>`_ on `github <https://github.com/bh107>`_ under the :ref:`LGPLv3 License <license>`.

If you want to join / contribute then fork the `repository <https://github.com/bh107/bohrium>`_  on Github and get in touch with us.

If you just want read-access then simply clone the repository::

  git clone git@github.com/bh107/bohrium.git
  cd bohrium

Continue by taking a look at :ref:`installation` on how to build / install Bohrium.
